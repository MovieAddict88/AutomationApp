# Baccarat Bot 2026

This project was converted from a decompiled APK into a buildable Android Studio project.

## Project Structure
- `app/`: Main application module.
  - `src/main/java/`: Reconstructed Kotlin source code.
  - `src/main/res/`: Reconstructed resources.
  - `src/main/jniLibs/`: Native libraries.
  - `src/main/assets/`: Application assets.
- `decompiled/`: Original decompiled files from `apktool` (smali and other metadata).

## How to Build
1. Open the project in Android Studio.
2. Sync the project with Gradle files.
3. Build and Run the `app` module.

## Notes
- The source code was reconstructed from smali and may have some simplifications compared to the original.
- The OCR logic in `MainActivityOptimized.kt` is a placeholder and may need further implementation if specific OCR behavior is required.
